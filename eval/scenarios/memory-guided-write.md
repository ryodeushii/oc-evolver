Use the oc-evolver plugin to create a reusable memory profile named `research-routing`.
The profile should route durable research, specs, and notes to Basic Memory by default.
Include `storage_mode: memory-and-artifact`.
Include `sources` for `memory://research/oc-evolver/*` and `memory://plans/oc-evolver/*`.
Include `queries` with `oc-evolver research routing`.
Do not write a repo-local spec, policy, or markdown file.
Stop after the memory profile exists.
